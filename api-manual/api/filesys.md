### Recursive

#### 详细描述
Recursive recursively walk through the file system

raw: the root path

opts: options

return: error



Example:
```
err := filesys.Recursive( //

	"testdata",
	filesys.dir(["cc", "dd"], filesys.onFileStat((name, info) => {})),

)
```


#### 定义

`Recursive(raw string, opts ...Option) error`

#### 参数
